The objective of this API is to give access to information about different movies, directors, and genres.

Tools: Javascript, HTML, Node.js, CSS and SQL

Request URL HTTP Method Response
Return a list of ALL movies to the user '/movies' GET Returns a JSON list of movies
Request data about a single movie by title to the user '/movies/:title' GET Returns JSON information about the movie requested
Return data about a genre by name/title '/genres/:genre' GET Returns JSON information about the movie genre requeted
Return data about a director by name '/directors/:directorName' GET Returns JSON information about the director requeted
Register New Users '/user/newUser' POST Adds a new User to JSON object
Update User Info 'users/:Username' PUT Returns the updated data fields by the user
Add a movie to list of favorites 'favorite/:movieName' POST Returns a text confirming which movie was added successfully
Delete a movie from the list of favorites /favorite/:deleteMovieName DELETE Returns a text confirming which movie was deleted successfully
Deregister a User 'users/:deleteUser' DELETE Returns a text confirming the user was deleted successfully
